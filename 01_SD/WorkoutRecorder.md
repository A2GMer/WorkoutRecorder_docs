| no          | 連番           | Integer   | ◯     |      |          | 
| record_date | 日付           | Timestamp | ◯     |      |          | 
| part        | 部位           | String    |        | ◯   |          | 
| event_name  | エクササイズ名 | String    |        | ◯   |          | 
| weight      | 重さ           | Double    |        | ◯   | 単位：KG |
